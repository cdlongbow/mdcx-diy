# 功能特色

MDCx 是一款功能强大、设计精良的视频元数据刮削和管理工具。以下是它的核心特色和创新点。

## 🎯 核心功能

### 智能刮削系统

**40+ 网站支持**

MDCx 支持从 40+ 个数据源网站自动获取视频元数据，覆盖各类内容类型：

| 类型 | 支持网站 | 特色 |
|------|---------|------|
| **有码** | DMM、MGStage、Prestige、Official、JavBus、Jav321、JavDB、JavDBAPI、MissAV、AVSOX、MMTV、MyWife、Getchu 等 | 覆盖主流有码网站，信息完整准确 |
| **无码** | Kin8、Love6 等 | 专业的无码内容刮削 |
| **FC2** | FC2、FC2Club、FC2Hub、FC2PPVDB | 完整的 FC2 番号支持 |
| **国产** | HDOUBAN、CNMDB、GUOCHAN、MADOUQU | 国产番号专项支持 |
| **欧美** | THEPORNDB | 欧美内容元数据 |
| **其他** | Giga、Faleno、Fantastica、Dahlia、XCity 等 | 优质内容补充 |

**智能番号识别**

自动识别和分类番号类型：
- 🇨🇳 **国产番号**：MD 系列、MKY 系列，自动标记为"国产"
- 👤 **素人番号**：SIRO 系列、短番号识别
- 🎥 **FC2 番号**：包含"FC2"的番号自动分类
- 🌍 **欧美番号**：格式 `XXX.00.00.00` 识别
- 🏷️ **特定网站**：KIN8、Getchu、MyWife 等特殊识别
- 🔞 **无码番号**：通过番号格式自动识别
- ✅ **默认处理**：有码番号的标准流程

**爬虫架构优势**

- 📦 **统一基类**：`GenericBaseCrawler` 提供标准化刮削流程
- 🔄 **上下文管理**：每次刮削任务独立 Context，避免状态污染
- ⚙️ **生命周期管理**：搜索页 → 详情页 → 数据提取 → 后处理
- 🔌 **可扩展性**：支持自定义 URL、隐藏在 UI 等高级功能
- 🎭 **多网站整合**：智能合并多个网站的结果，选择最佳数据

### 完整的元数据管理

**NFO 生成器**

生成完全符合 Kodi/Emby/Jellyfin 规范的 NFO 文件，包含 30+ 个元数据字段：

**基本信息**
- 番号、标题、原始标题、排序标题
- 剧情简介、标语

**发行信息**
- 首播日期、发行日期、年份
- 家长分级、自定义分级
- 国家代码

**人员信息**
- 演员列表（支持多语言名称）
- 导演列表

**评分信息**
- 公众评分、影评人评分
- 想看人数

**分类标签**
- 标签、类型、系列

**制作信息**
- 制作商、厂牌、发行商
- 标签

**媒体资源**
- 海报、缩略图、背景图
- 预告片 URL

**外部 ID**
- javdbid、javlibid 等各网站 ID

**数据模型**

完整的数据模型系统，包括：
- `FileInfo`：文件基础信息
- `CrawlerInput`：爬虫输入参数
- `CrawlTask`：刮削任务
- `CrawlersResult`：刮削结果
- `OtherInfo`：其他扩展信息

### 多语言翻译系统

**4 大翻译引擎**

1. **Google 翻译**：稳定可靠的 Google 翻译 API
2. **百度翻译**：支持中文优化的翻译服务
3. **DeepL 翻译**：高质量的机器翻译
4. **LLM 翻译**：支持自定义 LLM API（如 OpenAI、通义千问等）

**翻译特性**

- 📝 **字段级配置**：每个字段可独立配置翻译开关
- 🌐 **多语言支持**：中文简体、中文繁体、日语
- 📋 **映射表机制**：演员、制作商、标签等可通过映射表翻译
- 👤 **演员真实姓名**：从 Av-wiki 获取演员真实日文名
- 🔄 **降级策略**：翻译失败时保留原文或使用备用引擎

**翻译流程**

```
1. 语言检测
   ↓
2. 翻译引擎优先级尝试
   ↓
3. 应用翻译结果
   ↓
4. 简繁体转换（zhconv）
   ↓
5. 映射表应用
```
