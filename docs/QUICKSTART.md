# 5 分钟上手

第一次用 MDCx，按下面几步走，5 分钟就能刮出一部影片的信息。

## 第一步：准备工作

下载 Release 包（或从源码运行）后，打开程序会看到主界面。

## 第二步：设置媒体路径

1. 点菜单「设置」→「通用」
2. 在「媒体路径」右边点浏览，选你放视频文件的文件夹
3. 「成功输出目录」可选，设置后刮好的文件会复制一份过去

## 第三步：选刮削网站

1. 点「设置」→「刮削」
2. 在「有码网站」列表里勾选你想要的源（JavBus、JavDB 一般就够了）
3. 用上下箭头调整顺序，排在前面的优先用

如果刮的是无码、FC2、国产、欧美影片，在对应的标签页里设置。

## 第四步：开始刮削

1. 回到主界面，点「扫描」
2. 程序会列出文件夹里所有视频文件
3. 勾选你想处理文件（可以先选 1 个试试）
4. 点「开始」

等一会儿，你会看到日志区在跑数据。刮完后：
- 视频文件旁边会多一个同名的 `.nfo` 文件（里面是影片信息）
- 如果勾选了下载图片，还会多一个文件夹放海报、缩略图
- 文件会被重命名（按你设置的命名规则）

## 第五步：检查效果

用 Emby、Jellyfin 或 Kodi 打开媒体库，刷新一下，应该能看到多了一张海报墙。
