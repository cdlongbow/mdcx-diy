## 快速开始

### 第一次打开程序

1. 启动程序后，你会看到主界面
2. 程序会自动加载演员数据库（`userdata/actor_database.xlsx`）
3. 第一次用，建议先去检查设置

### 界面布局

- **菜单栏**：文件、编辑、工具、帮助
- **工具栏**：常用功能的快捷按钮
- **数据面板**：显示演员信息和搜索结果
- **状态栏**：显示当前操作和提示

## 主要功能

### 1. 管理演员数据库

演员数据存放在 `userdata/actor_database.xlsx`，直接用 Excel 操作：

- 查看和修改演员信息（日文名、中文名、繁体名、别名、信息链接、TMDB ID 等）
- 用 Excel 的筛选和搜索功能找演员
- 保存后程序下次启动会自动加载最新数据

### 2. 从 TMDB 获取数据

- 自动查找演员在 TMDB 上的 ID 和多语言信息
- 补全缺少的 TMDB ID
- 自动匹配和补全数据
- 控制请求速度，避免超过 TMDB 限制

### 3. 整理数据格式

- 自动清理无效数据
- 统一信息链接格式（LibreDMM 链接补全）
- 删除重复演员记录
- 格式化工作表

## 基本操作

### 搜索演员

1. 打开 `userdata/actor_database.xlsx`
2. 使用 Excel 筛选功能（数据 > 筛选）
3. 在筛选下拉菜单选择目标列（日文名、中文名、别名等），输入关键词搜索
4. 也可以按 Ctrl+F 在 Excel 中全局搜索

### 修改演员信息

1. 打开 `userdata/actor_database.xlsx`
2. 直接修改日文名、中文名、繁体名、别名、信息链接或 TMDB ID
3. 保存文件，程序下次启动会自动读取

### 备份数据

1. 关掉程序
2. 复制 `userdata/actor_database.xlsx` 到安全位置
3. 恢复时把备份文件复制回原位置

## 演员数据库管理

### 数据库字段

每条演员记录包含：
- **ID**：唯一编号
- **日文名**：演员的日文原名
- **中文名**：中文姓名
- **繁体名**：繁体中文姓名
- **别名**：其他叫法
- **信息链接**：相关网页链接（LibreDMM、TMDB 等）
- **TMDB ID**：TMDB 数据库编号

### 日常维护

#### 格式化数据库

1. 点菜单：工具 > 格式化数据库
2. 程序自动清理无效数据、统一链接格式、整理工作表
3. 查看格式化报告

#### 备份

建议定期复制 `userdata/actor_database.xlsx` 到安全位置保存。

## TMDB 数据刮削

### 刮削时自动补全信息

1. 在设置中填写 TMDB API Key
2. 开始刮削影片
3. 程序自动：
   - 检查演员数据库中的 TMDB ID
   - 优先使用本地数据库已有名字（日文名、中文名等）
   - 对缺少 TMDB ID 的演员调用 TMDB 查找
   - 将结果写回 `userdata/actor_database.xlsx`
4. 查看生成的 NFO 文件和演员数据库

### 配置 TMDB

- 去 [TMDB API](https://www.themoviedb.org/settings/api) 申请 API Key
- 在程序设置中输入 Key，保存

**注意**：程序内置限流保护，默认约每秒 3.5 次请求、3 路并发，避免被 TMDB 封禁。

## 故障排除

### 常见问题

#### 1. TMDB 提示请求太多

**现象**：频繁出现 429 Too Many Requests

**解决**：
- 降低并发数（建议设为 3）
- 开启令牌桶限流（默认每秒 3.5 次请求）
- 等一会儿再试

#### 2. 数据库加载失败

**现象**：启动时提示无法加载数据库

**解决**：
- 检查文件路径
- 确认文件未被其他程序占用
- 用备份文件恢复

#### 3. 搜索不到演员

**现象**：搜索演员无结果

**解决**：
- 检查搜索词是否准确
- 换别名或不同语言名字搜索
- 确认演员名和别名正确，必要时手动修改后重新刮削

#### 4. Linux 上界面显示不正常

**现象**：界面不显示或闪退

**解决**：
```bash
# 装缺失的系统库
sudo apt install libxcb-xinerama0 libxcb-cursor0

# 设环境变量
export QT_QPA_PLATFORM=xcb
```

#### 5. 高分屏界面过大或过小

**现象**：在 1920x1080 高分屏（如 125%/150% 缩放）上界面元素过大，窗口无法缩小到合适范围。

**解决**：
- 打开"设置" -> "界面外观" -> "高分屏缩放"
- 从下拉框选择合适的缩放比例（跟随系统 / 80% / 90% / 100% / 125% / 150% / 175% / 200%）
- 如果界面依然模糊，勾选"非整数倍缩放"复选框
- 保存配置后重启软件生效
