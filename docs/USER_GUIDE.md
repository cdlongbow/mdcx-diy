## 快速开始

### 第一次打开程序

1. 启动程序后，你会看到主界面
2. 程序会自动加载演员数据库（`userdata/actor_database.xlsx`）
3. 第一次用，建议先去检查一下设置

### 界面长什么样

主界面分几个区域：
- **菜单栏**：文件、编辑、工具、帮助
- **工具栏**：常用功能的快捷按钮
- **数据面板**：显示演员信息和搜索结果
- **状态栏**：显示当前在做的事情和提示

## 主要功能

### 1. 管理演员数据库

演员数据存在一个 Excel 文件里（`userdata/actor_database.xlsx`），你直接用 Excel 就能操作：

- 用 Excel 打开这个文件
- 查看和修改演员信息（日文名、中文名、繁体名、别名、信息链接、TMDB ID 等）
- 用 Excel 的筛选和搜索功能找演员
- 改完后保存，程序下次启动会自动加载最新数据

### 2. 从 TMDB 获取数据

TMDB 是一个电影数据库。这个功能可以帮你：
- 自动查找演员在 TMDB 上的 ID 和多语言信息
- 补全缺少的 TMDB ID
- 自动匹配和补全数据
- 控制请求速度，避免超过 TMDB 的限制

### 3. 整理数据格式

- 自动清理无效数据
- 统一信息链接的格式
- 删除重复的演员记录
- 优化存储结构

## 基本操作

### 搜索演员

演员数据在 Excel 文件里，搜索也直接在 Excel 里做：

1. 打开 `userdata/actor_database.xlsx`
2. 点 Excel 的筛选功能（数据 > 筛选）
3. 在筛选下拉菜单里，选你要搜哪一列（日文名、中文名、别名等）
4. 输入关键词就能快速找到演员

也可以按 Ctrl+F 在 Excel 里全局搜索。

### 修改演员信息

直接在 Excel 里改：

1. 打开 `userdata/actor_database.xlsx`
2. 找到要改的演员
3. 直接修改日文名、中文名、繁体名、别名、信息链接或 TMDB ID
4. 保存文件

程序下次启动时会自动读取最新数据。

### 备份数据

演员数据在 Excel 文件里，直接复制文件就能备份：

1. 关掉程序（确保文件没在被占用）
2. 把 `userdata/actor_database.xlsx` 复制到安全的地方
3. 需要恢复时，把备份文件复制回来替换就行

## 演员数据库管理

### 数据库有哪些信息

每条演员记录包含：
- **ID**：唯一编号
- **日文名**：演员的日文原名
- **中文名**：中文姓名
- **繁体名**：繁体中文姓名
- **别名**：其他叫法
- **信息链接**：相关网页链接（LibreDMM、JavDB、TMDB 等）
- **TMDB ID**：TMDB 数据库里的编号

### 日常维护

#### 整理数据库

定期整理可以让数据更干净：

1. 点菜单：工具 > 格式化数据库
2. 程序会自动做这些事：
   - 清理无效数据
   - 统一链接格式
   - 整理成统一格式
3. 看完整理报告就完成了

#### 备份

建议定期把 `userdata/actor_database.xlsx` 复制到安全的地方保存。

## TMDB 数据刮削

### 刮削时自动补全信息

1. 在设置里填上你的 TMDB API Key
2. 开始刮削影片
3. 程序会自动：
   - 检查演员数据库里的 TMDB ID
   - 优先用本地数据库已有的名字（日文名、中文名等）
   - 对没有 TMDB ID 的演员，调用 TMDB 查找
   - 把查到的结果写回 `userdata/actor_database.xlsx`
4. 查看生成的 NFO 文件和演员数据库

### 怎么配置 TMDB

- 去 [TMDB API](https://www.themoviedb.org/settings/api) 申请一个 API Key
- 在程序设置里输入这个 Key
- 保存设置就行

**注意**：程序内置了限流保护，默认约每秒 3.5 次请求、3 路同时查询，避免被 TMDB 封掉。

## 高级功能

### 从多个数据源补全信息

演员信息可以从好几个地方补全：

1. **TMDB API**：电影数据库
2. **Wikidata**：维基百科数据
3. **Gfriends**：演员头像和写真（支持本地仓库或 GitHub 镜像）
4. **graphis.ne.jp**：写真集信息

详细说明请看[TMDB 数据流文档](CODE_WIKI.md#tmdb-数据流)。

### 配合 Emby/Jellyfin 使用

可以和 Emby、Jellyfin 媒体服务器配合：
- 自动补全演员信息
- 获取演员简介
- 更新演员头像

配置方法请看[Emby 集成文档](CODE_WIKI.md#emby-集成)。

## 故障排除

### 常见问题

#### 1. TMDB 提示请求太多

**现象**：频繁出现 429 Too Many Requests 错误

**怎么办**：
- 降低并发数（建议设成 3）
- 开启令牌桶限流（默认每秒 3.5 次请求）
- 等一会儿再试

#### 2. 数据库加载失败

**现象**：启动程序时提示无法加载数据库

**怎么办**：
- 检查文件路径对不对
- 确认文件没被其他程序占用
- 试试用备份文件恢复

#### 3. 搜索不到演员

**现象**：搜演员时没有结果

**怎么办**：
- 检查搜索词有没有打错
- 换别名或不同语言的名字试试
- 检查演员名和别名是否正确，必要时手动改了再重新刮削

#### 4. Linux 上界面显示不正常

**现象**：界面不显示或闪退

**怎么办**：
```bash
# 装缺失的系统库
sudo apt install libxcb-xinerama0 libxcb-cursor0

# 设环境变量
export QT_QPA_PLATFORM=xcb
```
